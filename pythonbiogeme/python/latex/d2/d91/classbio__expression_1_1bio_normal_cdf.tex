\subsection{Detailed Description}
Class representing the cumulative distribution function of the normal distribution. 

The C\+D\+F of the normal distribution is \[ \mbox{bioNormalCdf}(x) = \frac{1}{\sqrt{2\pi}}\int_{-\infty}^x \exp\left(\frac{-t^2}{2}\right) dt \] A typical example is the probability of a binary probit model, which is computed as follows\+: 
\begin{DoxyCode}
1 prob = normalCdf(V1-V2)
\end{DoxyCode}
