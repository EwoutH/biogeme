16/** C++ Example NLP for interfacing a problem with IPOPT.
17 *  HS071_NLP implements a C++ example of problem 71 of the
18 *  Hock-Schittkowski test suite. This example is designed to go
19 *  along with the tutorial document and show how to interface
20 *  with IPOPT through the TNLP interface.
21 *
22 * Problem hs071 looks like this
23 *
24 *     min   x1*x4*(x1 + x2 + x3)  +  x3
25 *     s.t.  x1*x2*x3*x4                   >=  25
26 *           x1**2 + x2**2 + x3**2 + x4**2  =  40
27 *           1 <=  x1,x2,x3,x4  <= 5
28 *
29 *     Starting point:
30 *        x = (1, 5, 5, 1)
31 *
32 *     Optimal solution:
33 *        x = (1.00000000, 4.74299963, 3.82114998, 1.37940829)
34 *
35 *
36 */
