Detailed Description

Class implementing an iterator on a list of expressions. Note that, mathematically, a list of sums is equivalent to a sum of lists. But this class is designed to keep the structure of the sum, necessary to compute the BHHH, which requires the gradient for each element of the sum. This is typically required for a loglikelihood function, so the name.
